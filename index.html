DESIGNING

A NEW WEB

INTRODUCTION

• I decided to call my project “Designing A New Web” because I’m designing my own version of the web through a series of websites.

• My goal was to carve out my own spaces on the web by creating spaces that prioritise slowness, creativity, collaboration, and craft.

• How can I use the web as a medium to design my own ecosystem of experimental tools for community, collaboration, and creativity?

RESEARCH

• Started the semester with a lot of reading which shaped my conceptual flow throughout the past 12 weeks.

• J. R. Carpenter’s “A Handmade Web” resonated with me, especially the suggestion to slow down and find the value in creating websites with our hands, the way we would carefully craft printed design artefacts, instead of always relying on existing software.

• Other significant reading done was the book Entangled Life (got me thinking about plants and real life webs) and the Nature of Code book (used throughout the entire semester as a code reference).

RUBBER BAND ART

• Worked on this website for the first half of the semester and haven’t really touched it since, which looking back on it now allowed me to see things I want to improve, such as improving the speed of the website (it’s very slow right now) and re-thinking how the shapes are formed.

• The idea behind this website was inspired by rainbow looms and peg board rubber band art, but turned into a collaborative drawing board where "bands" are added permanently and anyone can drag them around to create abstract images.

• Learned a lot of technical skills for this and improved my visual development process for web design.

COMMUNITY GARDEN EXPLANATION

• Worked on this the entire semester, first half was planning and visual development, then second half was web development and a lot of adapting to make things work with how I imagined and within my coding abilities.

• I really wasn’t sure if I could finish it because by midterms, I had gotten so ambitious without actually doing any web development, so when I had to switch towards code it felt overwhelming.

• My goal with this project was to create a similar experience of being a part of a community garden but in an online space that can be used anywhere. I wanted to use my own interest in plants to create something that people would want to come back to and feel proud of their communal progress.

• Official name is “Botanical Bytes” (more catchy, possibly too alliterative) but I’ve been calling it the community garden for so long I can’t stop now.
